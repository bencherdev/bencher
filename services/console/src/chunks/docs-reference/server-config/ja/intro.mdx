Bencher APIサーバーは、起動時にJSON構成を必要とします。
構成は以下の3つの方法で提供可能です:
- `BENCHER_CONFIG`環境変数: 値はJSON構成に設定する必要があります
- `BENCHER_CONFIG_PATH`環境変数: 値はJSON構成を含むファイルのパスに設定する必要があります
- `/etc/bencher/bencher.json`ファイル: この場所にあるJSON構成を含むファイル

構成が見つからない場合、デフォルト構成が読み込まれます。
