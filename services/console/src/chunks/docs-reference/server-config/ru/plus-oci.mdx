### `plus.oci`

Этот раздел определяет конфигурацию хранилища реестра OCI (Open Container Initiative).
Bencher может функционировать как OCI-совместимый реестр контейнеров/артефактов.

Если этот раздел не настроен, Bencher будет использовать локальное файловое хранилище
в директории `oci` рядом с файлом базы данных.
Это подходит для разработки и развёртываний с одним экземпляром.

Для производственных развёртываний с несколькими экземплярами или для надёжности
настройте хранилище на базе S3.

Доступен один `service` хранилища:
- `aws_s3`: Хранение OCI-артефактов в AWS S3
    - `access_key_id`: Идентификатор ключа доступа AWS
    - `secret_access_key`: Секретный ключ доступа AWS
    - `access_point`: ARN точки доступа S3 с необязательным префиксом пути.
      Формат: `arn:aws:s3:<region>:<account-id>:accesspoint/<access-point-name>[/<path-prefix>]`

|             Имя              |                               Пример                                | По умолчанию | Обязательно |                                   Описание                                   |
| :--------------------------: | :-----------------------------------------------------------------: | :----------: | :---------: | :--------------------------------------------------------------------------: |
|          data_store          |                              \{ ... \}                              |     ---      |     Нет     |          Указывает бэкенд хранилища OCI. Локальный, если не указан.          |
|      data_store.service      |                              "aws_s3"                               |     ---      |     Да      |                       Указывает тип сервиса хранилища.                       |
|   data_store.access_key_id   |                       "AKIAIOSFODNN7EXAMPLE"                        |     ---      |     Да      |                       Идентификатор ключа доступа AWS.                       |
| data_store.secret_access_key |             "wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY"              |     ---      |     Да      |                         Секретный ключ доступа AWS.                          |
|   data_store.access_point    | "arn:aws:s3:us-east-1:123456789012:accesspoint/my-access-point/oci" |     ---      |     Да      | ARN точки доступа S3. Необязательный префикс пути после имени точки доступа. |
