### `--key <KEY>`

<br />

Opcional: La opción `--key` o la variable de entorno `BENCHER_API_KEY` puede establecerse con una clave de API de proyecto válida. Si ambas se especifican, la opción `--key` tiene precedencia sobre la variable de entorno `BENCHER_API_KEY`. Las claves de API de proyecto están limitadas a un solo proyecto y solo pueden usarse para `bencher run` y otros comandos no destructivos. Una clave de API de proyecto no puede usarse para crear un nuevo Proyecto ni para reclamar un Proyecto existente. Entra en conflicto con [la opción `--token`](#--token-token).
