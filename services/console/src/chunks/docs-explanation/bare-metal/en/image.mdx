## Image

An Image is an [OCI container image][oci image] that contains your benchmark code and its dependencies.
When a [Runner][runner] executes a Job, it pulls the Image, unpacks the Image, and executes the benchmark command specified in the [Job][job].
See [Bare Metal images][images] for a full overview.
