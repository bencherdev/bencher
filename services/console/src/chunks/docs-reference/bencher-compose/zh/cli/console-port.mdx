### `--console-port <PORT>`

为 Console 服务器容器指定端口号。
默认情况下使用端口 `3000`。
类似于 `docker run` 的 [仅端口形式的 `--expose` 选项][docker run expose]。

如果指定了 `--console-port` 选项，
您还必须更新 [API 服务器配置中的 `console.url`][console url] 以匹配新的端口，
可使用 [ `--api-env` 选项][api env] 或 [ `--api-volume` 选项][api volume]。
