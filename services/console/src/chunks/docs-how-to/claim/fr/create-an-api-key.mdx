## Créer une clé API

Afin d'utiliser la
<code><a href="/fr/docs/explanation/bencher-run/">bencher run</a></code> sous-commande CLI
avec [l'option `--key`][key option],
vous devrez créer une clé API utilisateur.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9mci9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">Cliquez pour ajouter une clé API</a></h3>

### Étapes :

1. Accédez à la console Bencher.
2. Survolez votre nom dans le coin supérieur droit.
3. Un menu déroulant devrait apparaître. Sélectionnez `Keys`.
4. Une fois sur la page des clés API, cliquez sur le bouton `➕ Add`.

[key option]: /fr/docs/explanation/bencher-run/#--key-key
