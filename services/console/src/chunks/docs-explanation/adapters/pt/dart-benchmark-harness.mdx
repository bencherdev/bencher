## 🎯 Dart benchmark_harness

O Adaptador Dart benchmark_harness (`dart_benchmark_harness`) espera a saída do pacote Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) (linhas padrão do `PrintEmitter`: `Name(RunTime): <microssegundos> us.`).
A Medida de `latência` (ou seja, `nanossegundos (ns)`) é coletada.
Apenas o valor reportado (ou seja, `valor`) está disponível. Não existem `valor_inferior` e `valor_superior`.
