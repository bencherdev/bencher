# Gungraun Example

To profile this example, you first have to install Valgrind and [`gungraun-runner`]:

```sh
sudo apt install valgrind
cargo install --version 0.18.1 gungraun-runner
```

[`gungraun-runner`]: https://gungraun.github.io/gungraun/latest/html/installation/gungraun.html#installation-of-the-benchmark-runner
