## `bencher mock`

`bencher mock` CLI 하위 명령은 모의 Bencher Metric Format (BMF) JSON 데이터를 생성하는 데 사용됩니다.

### `--seed <SEED>`

의사난수 생성기의 시드입니다.
지정하지 않으면 임의의 시드가 사용됩니다.

### `--count <COUNT>`

생성할 모의 [벤치마크][benchmark]의 수입니다.

[benchmark]: /ko/docs/explanation/benchmarking/#benchmark

### `--measure <MEASURE>`

각 [벤치마크][benchmark]에 사용할 [Measure][measure]입니다.
`--measure` 옵션을 여러 번 사용하여 여러 Measure를 지정할 수 있습니다.
기본값은 내장 `latency` Measure입니다.

[measure]: /ko/docs/explanation/benchmarking/#measure
[benchmark]: /ko/docs/explanation/benchmarking/#benchmark

### `--pow <POW>`

모의 [메트릭][metric]에 사용할 10의 거듭제곱입니다.
기본값은 `1`입니다.

[metric]: /ko/docs/explanation/benchmarking/#metric

### `--fail`

실행 중 실패하도록 합니다.

### `--flaky`

실행 중 간헐적으로 실패하도록 합니다.

### `--help`

도움말 메시지를 출력합니다.
