### `--spec-reset`

<br />

➕ Bencher Plus: [Testbed][testbed]の[Spec][spec]をリセットし、ハードウェア仕様を削除します。
実行が送信されると、Testbedに関連付けられた`spec`がクリアされます。
[`--testbed` オプション][testbed option]が必要です。
[`--image` オプション][image option]と競合します。
