## Image

An Image is an [OCI container image][oci image] that contains your benchmark code and its dependencies.
When a Runner executes a Job, it pulls the Image, unpacks the Image, and executes the benchmark command specified in the [Job][job].
See [the Image documentation][image] for how to create an Image for your own project.
