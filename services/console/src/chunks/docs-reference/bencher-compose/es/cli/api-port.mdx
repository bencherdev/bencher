### `--api-port <PORT>`

Especifica un número de puerto para el contenedor del servidor de la API.
Por defecto, se usa el puerto `61016`.
Similar a la forma que solo especifica el puerto de [la opción `--expose`][docker run expose] para `docker run`.

Si especificas la opción `--api-port`,
también debes actualizar [la variable de entorno `BENCHER_API_URL` del Console Server][console server]
para que coincida con el nuevo puerto,
usando la opción [la `--console-env`][console env].
