### `--key <KEY>`

<br />

Optionnel : L'option `--key` ou la variable d'environnement `BENCHER_API_KEY` peut être définie avec une clé API de projet valide. Si les deux sont spécifiées, l'option `--key` a la priorité sur la variable d'environnement `BENCHER_API_KEY`. Les clés API de projet sont limitées à un seul projet et ne peuvent être utilisées que pour `bencher run` et d'autres commandes non destructives. En conflit avec [l'option `--token`](#--token-token).
