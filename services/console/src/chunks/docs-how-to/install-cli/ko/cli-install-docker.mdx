## Docker

[Docker](https://docs.docker.com/engine/install/) 컨테이너를 사용하여 Bencher CLI를 실행하세요.

<CliDocker />

사용할 Bencher CLI의 정확한 버전을 지정하는 선택적인 버전 태그가 있습니다.
그렇지 않으면 최신 CLI 버전을 사용하는 것으로 기본 설정됩니다.
Bencher *Self-Hosted*를 사용하는 경우에만 정확한 버전을 지정하세요.
아직 끊임없는 주요 변경 사항이 있기 때문에 Bencher *Cloud*를 사용하는 경우에는 정확한 버전을 지정하지 __마세요__.
