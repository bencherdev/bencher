### `plus.google`

本节指定用于 [用于 OAuth2 身份验证的 Google 客户端](/zh/docs/how-to/google-oauth/) 的配置。您必须为服务器上的至少一个组织拥有有效的 Bencher Plus _企业版_ 许可证。整个部分为可选。如果未指定，则不会启用 Google 认证。

|     名称      |                           示例                           | 默认  | 必填  |                   描述                    |
| :-----------: | :------------------------------------------------------: | :---: | :---: | :---------------------------------------: |
|   client_id   | 0123456789-abcdefg0112358envs.apps.googleusercontent.com |  ---  |  是   | 指定您 Google OAuth2 客户端的客户端 ID。  |
| client_secret |                   GOCSPX-xyz987654321                    |  ---  |  是   | 用于您 Google OAuth2 客户端的客户端密钥。 |
