### `--job-timeout <SECONDS>`

<br />

Request a maximum [Job][job] execution time in seconds.
However, the following Job timeouts are always enforced with the provided concurrency limits:

| [Tier][pricing] | Job Timeout                     | Job Concurrency    |
| --------------- | ------------------------------- | ------------------ |
| Unclaimed       | 1 minute                        | 1 per source IP    |
| Free            | 5 minutes                       | 1 per Organization |
| Team            | Unlimited (default: 60 minutes) | Unlimited          |
| Enterprise      | Unlimited (default: 60 minutes) | Unlimited          |

This option requires [the `--image` option][image] to be set.
