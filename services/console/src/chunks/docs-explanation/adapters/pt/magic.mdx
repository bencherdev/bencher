## 🪄 Magic <small>(padrão)</small>

O Adaptador Magic (`magic`) é um superconjunto de todos os outros adaptadores.
Por esse motivo, ele é o adaptador padrão para `bencher run`.
No entanto, o adaptador `magic` deve ser usado apenas para exploração.
