### `plus.runners`

此部分配置用于裸机基准测试执行的运行器系统。
运行器是服务器范围的代理，负责认领和执行基准测试作业。

如果未配置此部分，Bencher 将使用默认值。

|           名称            | 示例 | 默认值 | 必需 |                                         描述                                          |
| :-----------------------: | :--: | :----: | :--: | :-----------------------------------------------------------------------------------: |
|    heartbeat_timeout      |  90  |   90   |  否  | 在没有心跳的情况下将作业标记为失败之前的时间（秒）。                                  |
| job_timeout_grace_period  |  60  |   60   |  否  | 超出作业配置的超时时间后服务器取消作业之前的额外时间（秒）。                          |
