### `--ci-only-on-alert`

<br />

Opcional: Solo comenzar a publicar resultados en CI si [se genera una Alerta](/es/docs/explanation/thresholds/#alerts). Si se genera una Alerta, entonces todos los resultados subsecuentes también se publicarán incluso si no contienen ninguna Alerta. Esto solo aplica a los comentarios de la solicitud de extracción. La Verificación de GitHub siempre se crea. Requiere: `--github-actions`
