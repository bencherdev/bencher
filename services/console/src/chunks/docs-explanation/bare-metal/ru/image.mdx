## Image

Image — это [OCI-образ контейнера][oci image], содержащий ваш код бенчмарка и его зависимости.
Когда Runner выполняет Job, он загружает Image, распаковывает Image и выполняет команду бенчмарка, указанную в [Job][job].
Смотрите [документацию Image][image] для получения информации о создании Image для вашего проекта.
