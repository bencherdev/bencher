### `--spec-reset`

<br />

Сбрасывает [Spec][spec] [Testbed][testbed], удаляя аппаратную спецификацию.
Это очищает `spec`, связанную с Testbed, при отправке запуска.
Требуется [опция `--testbed`][testbed option].
Конфликтует с [опцией `--image`][image option].
