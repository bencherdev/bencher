## Docker

Exécutez le Bencher CLI en utilisant un conteneur [Docker](https://docs.docker.com/engine/install/).

<CliDocker />

Il existe un tag de version optionnel pour spécifier une version précise du Bencher CLI à utiliser.
Sinon, il utilisera par défaut la dernière version de CLI.
Spécifiez une version précise si et seulement si vous utilisez Bencher *Self-Hosted*.
Ne __spécifiez__ pas une version précise si vous utilisez Bencher *Cloud* car il y a encore des changements occasionnels qui peuvent causer des problèmes.
