`runner` バイナリは Bare Metal ベンチマーク Job を実行するエージェントです。
[セルフホスト型 Runner][self hosted runners] をセットアップするための詳細なリファレンスです。
このバイナリは 2 つのサブコマンドを提供します。

- [`runner up`][runner up]: Runner を起動し、ベンチマーク Job をポーリングして実行します。
- [`runner run`][runner run]: Image をプルし、テスト用にローカルホスト上で 1 回だけ実行します。

> 🐰 `runner` バイナリは `bencher runner` サブコマンドとは異なります。
> `runner` バイナリは Runner ホストを運用するのに対し、
> [`bencher runner`][runners api] は REST API を通じて Runner リソースを管理します。
