### `plus.rate_limiting`

本节指定 API 服务器的速率限制。
整节内容是可选的。
如果未指定，将使用列出的默认值。

为防止创建[`unclaimed`项目][unclaimed projects]，请将`unclaimed_limit`设置为`0`。

|          名称          |      示例      |    默认值    | 是否必需 |                                                 描述                                                  |
| :--------------------: | :----------: | :----------: | :------: | :---------------------------------------------------------------------------------------------------: |
|         window         |    86400     |    86400     |    否    |                            指定以秒为单位的速率限制窗口。默认设置为一天。                             |
|    unclaimed_limit     |     255      |     255      |    否    | 控制[`unclaimed`项目][unclaimed projects]的速率限制。为了防止创建`unclaimed`项目，请将此值设置为`0`。 |
|     claimed_limit      |    65536     |    65536     |    否    |                                     控制`claimed`项目的速率限制。                                     |

#### `public`

`public`速率限制器按IP地址应用于未认证的请求。
每个子字段包含`minute`、`hour`和`day`限制。
整节内容是可选的。
如果未指定，将使用列出的默认值。

|       名称       |   示例   | 默认值 | 是否必需 |                           描述                                          |
| :--------------: | :-----: | :----: | :------: | :---------------------------------------------------------------------: |
|     requests     | \{ ... \} |  ---  |    否    | 指定每个IP地址的请求速率限制。                                            |
| requests.minute  |  1024   | 1024  |    否    | 每个IP地址每分钟最大请求数。                                              |
|  requests.hour   |  4096   | 4096  |    否    | 每个IP地址每小时最大请求数。                                              |
|   requests.day   |  8192   | 8192  |    否    | 每个IP地址每天最大请求数。                                                |
|     attempts     | \{ ... \} |  ---  |    否    | 指定每个IP地址的认证尝试速率限制。                                        |
| attempts.minute  |   32    |  32   |    否    | 每个IP地址每分钟最大认证尝试次数。                                        |
|  attempts.hour   |   64    |  64   |    否    | 每个IP地址每小时最大认证尝试次数。                                        |
|   attempts.day   |   128   |  128  |    否    | 每个IP地址每天最大认证尝试次数。                                          |
|       runs       | \{ ... \} |  ---  |    否    | 指定未认领项目的每个IP地址基准测试运行速率限制。                            |
|   runs.minute    |   64    |  64   |    否    | 每个IP地址每分钟最大基准测试运行数。                                      |
|    runs.hour     |   128   |  128  |    否    | 每个IP地址每小时最大基准测试运行数。                                      |
|     runs.day     |   256   |  256  |    否    | 每个IP地址每天最大基准测试运行数。                                        |
