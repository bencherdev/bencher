### `--api-port <PORT>`

Укажите номер порта для контейнера сервера API.
По умолчанию используется порт `61016`.
Эквивалентно форме указания только порта опции [`--expose`][docker run expose] для `docker run`.

Если вы указываете опцию `--api-port`, вы также должны обновить [переменную окружения Console Server `BENCHER_API_URL`][console server], чтобы она соответствовала новому порту, используя [опцию `--console-env`][console env].
