## Benchmark
Ein Benchmark ist ein benannter Leistungsregressionstest.
Wenn der Leistungsregressionstest neu für Bencher ist, wird automatisch ein Benchmark erstellt.
Andernfalls wird der Name des Leistungsregressionstests als eindeutiger Bezeichner für den Benchmark verwendet.

Seien Sie vorsichtig, wenn Sie den Namen Ihrer Leistungsregressionstests ändern.
Sie müssen den Benchmark in Bencher manuell umbenennen, damit er mit dem neuen Namen übereinstimmt.
Andernfalls wird der umbenannte Leistungsregressionstest als neuer Benchmark angesehen.
Diese Warnung gilt auch, wenn Sie einige Leistungsregressionstests _verschieben_.
Je nach Benchmark‑Harness kann der Pfad zum Leistungsregressionstest ein Teil seines Namens sein.

Die einzige Ausnahme zu obiger Warnung ist das Ignorieren eines Benchmarks.
Siehe [Unterdrückung von Warnungen][suppressing alerts] für eine vollständige Übersicht.

Wenn Sie das [Bencher Metric Format (BMF)][bmf] verwenden, können Name, Slug oder UUID zur Identifizierung eines Benchmarks verwendet werden.
