### `--ci-only-thresholds`

<br />

Optionnel : Publier les résultats sur CI uniquement si [un Seuil existe](/fr/docs/explanation/thresholds/) pour la Branche, l'Environnement de test et la Mesure.
S'il n'existe aucun Seuil, alors aucun commentaire de pull request ne sera publié.
Cela ne s'applique qu'aux commentaires de pull request. La vérification GitHub est toujours créée.
Requis : `--github-actions`
