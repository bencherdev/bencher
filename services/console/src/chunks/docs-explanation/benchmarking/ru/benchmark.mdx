## Бенчмарк
Бенчмарк — это именованный тест регрессии производительности.
Если тест регрессии производительности новый для Bencher, то Бенчмарк создаётся автоматически.
В противном случае в качестве уникального идентификатора Бенчмарка используется имя теста регрессии производительности.

Будьте осторожны при изменении имени ваших тестов регрессии производительности.
Вам придётся вручную переименовать Бенчмарк в Bencher, чтобы он совпадал с новым именем.
В противном случае переименованный тест регрессии производительности будет считаться новым Бенчмарком.
То же предостережение относится и к _перемещению_ некоторых тестов регрессии производительности.
В зависимости от фреймворка для запуска бенчмарков, путь к тесту регрессии производительности может быть частью его имени.

Единственным исключением из приведённого выше предостережения является игнорирование Бенчмарка.
См. [подавление оповещений][suppressing alerts] для полного обзора.

Если вы используете [Формат метрик Bencher (BMF)][bmf], для идентификации Бенчмарка можно использовать имя, slug или UUID.
