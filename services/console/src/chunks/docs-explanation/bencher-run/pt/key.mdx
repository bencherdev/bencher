### `--key <KEY>`

<br />

Opcional: A opcao `--key` ou a variavel de ambiente `BENCHER_API_KEY` pode ser definida com uma chave de API de projeto valida. Se ambas forem especificadas, a opcao `--key` tem precedencia sobre a variavel de ambiente `BENCHER_API_KEY`. As chaves de API de projeto sao limitadas a um unico projeto e so podem ser usadas para `bencher run` e outros comandos nao destrutivos. Conflita com [a opcao `--token`](#--token-token).
