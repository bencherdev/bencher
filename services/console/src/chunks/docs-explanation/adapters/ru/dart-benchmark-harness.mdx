## 🎯 Dart benchmark_harness

Адаптер Dart benchmark_harness (`dart_benchmark_harness`) ожидает вывод пакета Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) (стандартные строки `PrintEmitter`: `Name(RunTime): <микросекунды> us.`).
Собирается мера `latency` (т.е. `наносекунды (ns)`).
Доступно только зарегистрированное значение (т.е. `value`). Нет `lower_value` и `upper_value`.
