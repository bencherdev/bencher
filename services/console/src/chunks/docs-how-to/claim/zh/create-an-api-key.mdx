## 创建 API 密钥

为了使用 <code><a href="/zh/docs/explanation/bencher-run/">bencher run</a></code> CLI 子命令与 [`--key` 选项][key option]，您需要创建一个用户 API 密钥。

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi96aC9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">点击添加 API 密钥</a></h3>

### 步骤：

1. 前往 Bencher 控制台。
2. 将鼠标悬停在右上角的您的名字上。
3. 应出现一个下拉菜单。选择 `Keys`。
4. 在 API 密钥页面，点击 `➕ Add` 按钮。

[key option]: /zh/docs/explanation/bencher-run/#--key-key
