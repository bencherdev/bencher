## `bencher mock`

`bencher mock` CLI 子命令用于生成模拟 Bencher 指标格式（BMF）的 JSON 数据。

### `--seed <SEED>`

用于伪随机数生成器的种子。如果未指定，则会使用随机种子。

### `--count <COUNT>`

要生成的模拟[基准][benchmark]的数量。

[benchmark]: /zh/docs/explanation/benchmarking/#benchmark

### `--measure <MEASURE>`

要为每个[基准][benchmark]使用的[度量][measure]。可以通过多次使用 `--measure` 选项来指定多个度量。默认是内置的 `latency` 度量。

[measure]: /zh/docs/explanation/benchmarking/#measure
[benchmark]: /zh/docs/explanation/benchmarking/#benchmark

### `--pow <POW>`

用于模拟[度量指标][metric]的 10 的幂。默认值为 `1`。

[metric]: /zh/docs/explanation/benchmarking/#metric

### `--fail`

在运行时失败。

### `--flaky`

在运行时间歇性失败。

### `--help`

打印帮助信息。
