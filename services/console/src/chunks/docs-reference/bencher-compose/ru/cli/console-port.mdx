### `--console-port <PORT>`

Укажите номер порта для контейнера сервера Console.
По умолчанию используется порт `3000`.
Аналогично форме, задающей только порт, у [опции `--expose`][docker run expose] для `docker run`.

Если вы укажете опцию `--console-port`,
вам также необходимо обновить [конфигурацию API‑сервера `console.url`][console url],
чтобы она соответствовала новому порту,
используя либо опцию [ `--api-env` ][api env], либо опцию [ `--api-volume` ][api volume].
