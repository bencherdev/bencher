### `--console-port <PORT>`

Console 서버 컨테이너에 대한 포트 번호를 지정합니다.
기본적으로 포트 `3000`이 사용됩니다.
이는 `docker run`의 [`--expose` 옵션][docker run expose]의 포트 전용 형식과 유사합니다.

`--console-port` 옵션을 지정한 경우, API 서버의 `console.url` 구성을 새 포트에 맞게 업데이트해야 합니다. 이는 [`--api-env` 옵션][api env] 또는 [`--api-volume` 옵션][api volume] 중 하나를 사용하여 수행할 수 있습니다.
