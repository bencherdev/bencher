Самый простой способ развернуть [Bencher Self-Hosted][bencher self hosted] — использовать триптих подкоманд `bencher` CLI:

- [`bencher up`][bencher up]: Поднять контейнеры Bencher Self-Hosted.
  Это похоже на выполнение команды [`docker compose up`][docker compose up].
- [`bencher logs`][bencher logs]: Просмотр вывода из контейнеров Bencher Self-Hosted,
  если `bencher up` была выполнена с использованием [флага `--detach`][detach flag].
  Это похоже на выполнение команды [`docker compose logs`][docker compose logs].
- [`bencher down`][bencher down]: Остановить и удалить контейнеры Bencher Self-Hosted,
  если `bencher up` была выполнена с использованием [флага `--detach`][detach flag].
  Это похоже на выполнение команды [`docker compose down`][docker compose down].
