## Fluxo de Trabalho do Self-Hosted Runner

Uma vez que o Runner está conectado, você envia benchmarks da mesma forma que com qualquer Bare Metal Runner:
construa e publique uma [Image][images], depois execute [`bencher run --image`][images] com uma Spec correspondente.
