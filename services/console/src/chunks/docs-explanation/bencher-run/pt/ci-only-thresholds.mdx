### `--ci-only-thresholds`

<br />

Opcional: Somente publique resultados no CI se [um Limite existir](/pt/docs/explanation/thresholds/) para o Branch, Testbed e Medida.
Se não existirem Limites, nenhum comentário no pull request será publicado.
Isso se aplica apenas aos comentários no pull request. A Verificação do GitHub é sempre criada.
Requer: `--github-actions`
