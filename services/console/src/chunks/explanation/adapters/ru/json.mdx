## \{...\} JSON

Адаптер JSON (`json`) работает с JSON в формате [Bencher Metric Format (BMF)][bmf].
Он идеально подходит для интеграции пользовательских средств измерения производительности с Bencher.
Для более подробной информации смотрите [как отслеживать пользовательские бенчмарки][custom benchmarks]
и [ссылку на BMF JSON][bmf].
