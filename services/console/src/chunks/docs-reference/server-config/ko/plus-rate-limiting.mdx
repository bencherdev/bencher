### `plus.rate_limiting`

이 섹션은 API 서버에 대한 속도 제한을 지정합니다. 전체 섹션은 선택 사항입니다. 지정하지 않으면 나열된 기본값이 사용됩니다.

[`unclaimed` 프로젝트][unclaimed projects]의 생성을 방지하려면, `unclaimed_limit`을 `0`으로 설정하십시오.

|          이름          |      예      |    기본값    | 필수 항목 |                                                                         설명                                                                          |
| :--------------------: | :----------: | :----------: | :-------: | :---------------------------------------------------------------------------------------------------------------------------------------------------: |
|         window         |    86400     |    86400     |  아니오   |                                          초 단위의 속도 제한 창을 지정합니다. 기본적으로 하루로 설정됩니다.                                           |
|    unclaimed_limit     |     255      |     255      |  아니오   | [`unclaimed` 프로젝트][unclaimed projects]에 대한 속도 제한을 제어합니다. `unclaimed` 프로젝트가 생성되지 않도록 하려면 이 값을 `0`으로 설정하십시오. |
|     claimed_limit      |    65536     |    65536     |  아니오   |                                                   `claimed` 프로젝트에 대한 속도 제한을 제어합니다.                                                   |
