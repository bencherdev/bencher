## Benchmark
Un Benchmark est un test de régression de performance nommé.
Si le test de régression de performance est nouveau pour Bencher, un Benchmark est créé automatiquement.
Sinon, le nom du test de régression de performance est utilisé comme identifiant unique du Benchmark.

Faites attention lorsque vous changez le nom de vos tests de régression de performance.
Vous devrez renommer manuellement le Benchmark dans Bencher pour qu'il corresponde à ce nouveau nom.
Sinon, le test de régression de performance renommé sera considéré comme un nouveau Benchmark.
Cette même mise en garde s'applique également au fait de _déplacer_ certains tests de régression de performance.
Selon le framework de benchmark, le chemin vers le test de régression de performance peut faire partie de son nom.

La seule exception à la mise en garde ci‑dessus consiste à ignorer un Benchmark.
Consultez la [suppression des alertes][suppressing alerts] pour une vue d'ensemble complète.

Si vous utilisez le [Bencher Metric Format (BMF)][bmf], le nom, le slug ou l'UUID peuvent servir à identifier un Benchmark.
