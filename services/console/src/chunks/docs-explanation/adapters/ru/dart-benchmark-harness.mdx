## 🎯 Dart benchmark_harness

The Dart benchmark_harness Adapter (`dart_benchmark_harness`) expects output from the Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) package (default `PrintEmitter` lines: `Name(RunTime): <microseconds> us.`).
The `latency` Measure (ie `nanoseconds (ns)`) is gathered.
Only the reported value (ie `value`) is available.
Neither `lower_value` nor `upper_value` are collected.
