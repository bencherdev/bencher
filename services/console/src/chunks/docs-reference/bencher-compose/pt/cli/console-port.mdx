### `--console-port <PORT>`

Especifica um número de porta para o contêiner do servidor Console.
Por padrão, a porta `3000` é usada.
Semelhante à forma que expõe apenas a porta da [opção `--expose`][docker run expose] do `docker run`.

Se você especificar a opção `--console-port`,
também deverá atualizar [a configuração do servidor de API `console.url`][console url]
para corresponder à nova porta,
usando [a opção `--api-env`][api env] ou [a opção `--api-volume`][api volume].
