## 🎯 Dart benchmark_harness

Dart benchmark_harness 适配器（`dart_benchmark_harness`）期望的输出来自 Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) 包（默认 `PrintEmitter` 行：`Name(RunTime): <微秒> us.`）。
收集的是`延迟`度量（即`纳秒 (ns)`）。
只提供报告值（即`值`），没有`最小值`和`最大值`。
