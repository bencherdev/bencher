### `--key <KEY>`

<br />

Optional: Entweder die Option `--key` oder die Umgebungsvariable `BENCHER_API_KEY` kann auf einen gültigen Projekt-API-Schlüssel gesetzt werden. Wenn beide angegeben sind, hat die Option `--key` Vorrang vor der Umgebungsvariable `BENCHER_API_KEY`. Projekt-API-Schlüssel sind auf ein einzelnes Projekt beschränkt und können nur für `bencher run` und andere nicht-destruktive Befehle verwendet werden. Steht in Konflikt mit [der `--token` Option](#--token-token).
