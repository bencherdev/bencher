## Rastrear seus Benchmarks Bare Metal

Agora voce esta pronto para rastrear seus resultados de benchmark em hardware dedicado!
Para isso, voce usara [o subcomando CLI `bencher run`][bencher run]
com as opcoes `--project`, `--image` e `--spec`
para executar seus benchmarks em um Runner Bare Metal e coletar os resultados.

Usaremos o subcomando [`bencher noise`][bencher noise] como o comando de benchmark.
Ele mede o ruido do ambiente no hardware do Runner
e gera resultados no [Bencher Metric Format (BMF)][bmf] com `--format json`.

Usaremos o [Spec][spec] `intel-v1` para direcionar a um Runner Bare Metal baseado em Intel.
Consulte a [pagina de Spec][spec] para uma lista completa de Specs disponiveis e seus detalhes de hardware.
Substitua `YOUR_PROJECT_SLUG` pelo slug do projeto dos passos anteriores e execute:

<TerminalCommand code="bencher run --project YOUR_PROJECT_SLUG --image YOUR_PROJECT_SLUG:latest --spec intel-v1 bencher noise --format json" />

A opcao `--project` seleciona o projeto para rastrear os resultados,
a opcao `--image` define a imagem OCI para execucao remota,
e a opcao `--spec` seleciona o Spec de hardware para o Runner.
Tudo apos as opcoes do `bencher run` e o comando de benchmark que e executado dentro do container.

Agora voce pode visualizar os resultados de cada um dos seus benchmarks no navegador.
Clique ou copie e cole os links de `View results`.
