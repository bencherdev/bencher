### `--spec-reset`

<br />

➕ Bencher Plus: Restablece la [Spec][spec] del [Testbed][testbed], eliminando su especificación de hardware.
Esto borra la `spec` asociada al Testbed cuando se envía la ejecución.
Requiere: `--testbed`
Conflictos con: `--image`
