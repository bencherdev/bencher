#### Startpunkt
Ein Branch kann einen [Startpunkt][start point] haben.
Ein Startpunkt ist ein anderer Branch zu einer spezifischen Version (und `git` Hash, falls verfügbar).
Historische Metriken und optional Schwellenwerte werden vom Startpunkt übernommen.
Eine vollständige Übersicht finden Sie unter [Branches](/de/docs/explanation/branches/).

[start point]: https://git-scm.com/docs/git-branch#Documentation/git-branch.txt-ltstart-pointgt
