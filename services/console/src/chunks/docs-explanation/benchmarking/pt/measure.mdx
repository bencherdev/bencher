### Medida
Uma Medida é a unidade de medição para uma Métrica.
Por padrão, todos os Projetos começam com as Medidas `Latency` e `Throughput`, com unidades de `nanoseconds (ns)` e `operations / second (ops/s)`, respectivamente.
A Medida é determinada pelo [adaptador do harness de benchmark][adapters].

Se você estiver usando o [Bencher Metric Format (BMF)][bmf], o nome, o slug ou o UUID podem ser usados para identificar uma Medida.
