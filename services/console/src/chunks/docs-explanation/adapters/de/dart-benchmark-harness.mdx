## 🎯 Dart benchmark_harness

Der Dart benchmark_harness Adapter (`dart_benchmark_harness`) erwartet die Ausgabe des Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) Pakets (Standard `PrintEmitter` Zeilen: `Name(RunTime): <Mikrosekunden> us.`).
Das `latency` Maß (d.h. `Nanosekunden (ns)`) wird erfasst.
Nur der gemeldete Wert (d.h. `value`) ist verfügbar. Es gibt keinen `lower_value` und `upper_value`.
