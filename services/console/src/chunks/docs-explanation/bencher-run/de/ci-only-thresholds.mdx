### `--ci-only-thresholds`

<br />

Optional: Veröffentlicht Ergebnisse nur in CI, wenn [ein Schwellenwert existiert](/de/docs/explanation/thresholds/) für den Branch, das Testbett und die Messung.
Wenn keine Schwellenwerte existieren, wird kein Pull-Request-Kommentar veröffentlicht.
Dies gilt nur für Pull-Request-Kommentare. Die GitHub-Prüfung wird immer erstellt.
Erfordert: `--github-actions`
