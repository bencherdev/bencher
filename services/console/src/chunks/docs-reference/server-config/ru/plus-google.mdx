### `plus.google`

В этом разделе задаётся конфигурация для [Google-клиента, используемого для аутентификации через OAuth2](/ru/docs/how-to/google-oauth/).
У вас должна быть действительная лицензия Bencher Plus _Enterprise_ хотя бы для одной организации на сервере.
Весь раздел является необязательным.
Если раздел не указан, аутентификация через Google не будет включена.

|     Name      |                         Example                          | Default | Required |                            Description                            |
| :-----------: | :------------------------------------------------------: | :-----: | :------: | :---------------------------------------------------------------: |
|   client_id   | 0123456789-abcdefg0112358envs.apps.googleusercontent.com |   ---   |    Да    | Указывает идентификатор клиента для вашего клиента Google OAuth2. |
| client_secret |                   GOCSPX-xyz987654321                    |   ---   |    Да    |         Секрет клиента для вашего клиента Google OAuth2.          |
