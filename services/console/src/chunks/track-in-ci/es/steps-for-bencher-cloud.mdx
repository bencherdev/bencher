## Pasos para Bencher Cloud

1. [Crea una cuenta en Bencher Cloud][bencher sign up].
2. [Crea una clave de API de usuario][console users keys add] y agrégala a tu CI como un secreto.
3. Crea un flujo de trabajo para tu CI, como [GitHub Actions][github actions] o [GitLab CI/CD][gitlab ci cd].
4. [Instala el Bencher CLI][install bencher cli] en tu flujo de trabajo de CI.
