### Medida
Una Medida es la unidad de medida para una Métrica.
Por defecto todos los Proyectos empiezan con una Medida `Latency` y `Throughput` con unidades de `nanoseconds (ns)` y `operations / second (ops/s)` respectivamente.
La Medida la determina el [adaptador del arnés de benchmark][adapters].

Si estás usando [Bencher Metric Format (BMF)][bmf] el nombre, slug o UUID pueden usarse para identificar una Medida.
