#### Spec
Uma Spec é uma especificação de hardware que descreve os recursos disponíveis para um Testbed.
Inclui a arquitetura de CPU, número de CPUs, tamanho de memória, tamanho de disco e se o ambiente tem acesso à rede.
As Specs têm escopo de servidor, portanto a mesma Spec pode ser compartilhada entre múltiplos Testbeds.
