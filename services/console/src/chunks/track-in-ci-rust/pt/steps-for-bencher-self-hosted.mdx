## Passos para Bencher Self-Hosted

1. Crie uma [instância Bencher Self-Hosted][bencher self hosted].
2. Crie uma conta em sua instância Bencher Self-Hosted.
3. Crie um token de API e adicione-o ao seu CI como um segredo.
4. Crie um fluxo de trabalho para o seu CI, como [GitHub Actions][github actions] ou [GitLab CI/CD][gitlab ci cd].
5. [Instale o Bencher CLI][install bencher cli] no fluxo de trabalho do seu CI.
   Certifique-se de que a versão do CLI corresponda à versão da sua instância Bencher Self-Hosted.
6. Execute seus benchmarks com o
   <code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code>
   subcomando no fluxo de trabalho do seu CI usando
   <a href={props.url}>o adaptador <code>{props.adapter}</code></a>
   e configure a [opção `--host`][host option] para a URL da sua instância Bencher Self-Hosted.
