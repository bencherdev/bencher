## Назначение Spec для Runner

Runner Spec связывает [Spec][spec] с Runner.
Runner может поддерживать несколько Spec, а Spec может поддерживаться несколькими Runner.
Runner берёт только те Job, чей Spec ему назначен.

Добавьте Spec к Runner с помощью [CLI `bencher`][bencher runner spec cli]
или [REST API для Runner Specs][runner specs api]:

```sh
bencher runner spec add my-runner \
  --host https://api.bencher.example.com \
  --spec intel-v1
```

Удалить связь позже можно с помощью [API `DELETE /v0/runners/{runner}/specs/{spec}`][delete runner spec api].
