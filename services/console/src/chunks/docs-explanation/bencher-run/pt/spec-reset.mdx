### `--spec-reset`

<br />

➕ Bencher Plus: Redefine a [Spec][spec] do [Testbed][testbed], removendo sua especificação de hardware.
Isso limpa a `spec` associada ao Testbed quando a execução é enviada.
Requer: `--testbed`
Conflitos com: `--image`
