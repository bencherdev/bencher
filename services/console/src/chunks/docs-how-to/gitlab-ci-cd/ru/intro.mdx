В зависимости от вашего варианта использования, вы можете настроить [Непрерывное Бенчмарк-тестирование][continuous benchmarking] в GitLab CI/CD для:

- [Целевой ветки](#target-branch)
- [Запросов на слияние](#merge-requests)

Убедитесь, что вы создали ключ API проекта
и задали его как переменную с маской под именем `BENCHER_API_KEY` перед тем, как продолжить!
Перейдите в `Ваш репозиторий -> Настройки -> CI/CD -> Переменные -> Развернуть -> Добавить переменную`.
Ключ переменной должен быть `BENCHER_API_KEY`, а значение переменной должно быть вашим ключом API проекта.
Отметьте оба поля `Защитить переменную` и `Скрыть переменную`.
