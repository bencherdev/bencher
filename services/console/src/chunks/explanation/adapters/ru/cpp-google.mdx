## ➕ C++ Google

Адаптер C++ Google (`cpp_google`) ожидает вывод [Google Benchmark](https://github.com/google/benchmark) в [формате JSON (т.е. `--benchmark_format=json`)](https://github.com/google/benchmark/blob/main/docs/user_guide.md#output-formats).
Измеряется `latency` (т.е. наносекунды (ns)).
Доступно только среднее значение (т.е. `value`). `lower_value` и `upper_value` отсутствуют.
