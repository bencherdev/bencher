### `--console-port <PORT>`

Especifica un número de puerto para el contenedor del servidor Console.
Por defecto se usa el puerto `3000`.
Similar a la forma solo de puerto de [la opción `--expose`][docker run expose] para `docker run`.

Si especifica la opción `--console-port`,
también debe actualizar la [configuración del servidor API `console.url`][console url]
para que coincida con el nuevo puerto,
usando ya sea la opción [la opción `--api-env`][api env] o la opción [la opción `--api-volume`][api volume].
