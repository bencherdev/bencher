### `--ci-only-thresholds`

<br />

Opcional: Solo enviar resultados a CI si [existe un Umbral](/es/docs/explanation/thresholds/) para la Rama, Testbed y Medida. Si no existen Umbrales, no se publicará ningún comentario de la solicitud de extracción. Esto solo aplica a los comentarios de la solicitud de extracción. La Verificación de GitHub siempre se crea. Requiere: `--github-actions`
