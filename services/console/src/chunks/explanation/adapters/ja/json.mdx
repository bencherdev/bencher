## \{...\} JSON

JSON アダプタ（`json`）は、[Bencher Metric Format (BMF) JSON][bmf]を期待しています。
これは、カスタムベンチマークハーネスをBencherと統合するのに最適です。
詳細については、[カスタムベンチマークの追跡方法][custom benchmarks]と
[BMF JSON リファレンス][bmf]をご覧ください。
