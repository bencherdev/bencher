## Create an API Key

In order to use the
<code><a href="/docs/explanation/bencher-run/">bencher run</a></code> CLI subcommand
with [the `--key` option][key option],
you will need to create a user API key.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">Click to add an API Key</a></h3>

### Steps:

1. Navigate to the Bencher Console.
2. Hover over your name in top right corner.
3. A dropdown menu should appear. Select `Keys`.
4. Once on the API Keys page, click the `➕ Add` button.

[key option]: /docs/explanation/bencher-run/#--key-key
