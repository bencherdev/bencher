## \{...\} JSON

L'adaptateur JSON (`json`) attend du JSON au format [Bencher Metric Format (BMF)][bmf].
Il est parfait pour intégrer des harnais de benchmark personnalisés avec Bencher.
Pour plus de détails, consultez [comment suivre des benchmarks personnalisés][benchmarks personnalisés]
et la [référence JSON BMF][bmf].
