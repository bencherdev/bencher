## Bencher с собственным хостингом

Bencher является открытым исходным кодом и может быть размещён самостоятельно.
Если вы заинтересованы в использовании Bencher Cloud, ознакомьтесь с [руководством по быстрому старту Bencher Cloud][quickstart].
Это руководство поможет вам настроить использование Bencher с собственным хостингом с помощью Docker.

> 🐰 Когда вы почувствуете себя комфортно, используя Bencher с собственным хостингом, рассмотрите возможность изучения следующих ресурсов:
> - [Bencher с собственным хостингом][bencher self hosted]
> - [Конфигурация консольного сервера][console server config]
> - [Конфигурация API сервера][api server config]
> - [Архитектура Bencher][bencher architecture]
