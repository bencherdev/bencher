## 🎯 Dart benchmark_harness

Dart benchmark_harnessアダプター (`dart_benchmark_harness`) は Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) パッケージの出力（デフォルトの `PrintEmitter` 行: `Name(RunTime): <マイクロ秒> us.`）を期待しています。
`latency` 測定値（すなわち `ナノ秒 (ns)`）が収集されます。
報告された値（すなわち `value`）のみが利用可能です。`lower_value` と `upper_value` はありません。
