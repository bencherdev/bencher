### `--ci-on-the-fly`

<br />

Opcional: Permitir la creación de [Proyectos Sobre la Marcha][on the fly projects] en entornos CI. Requerido si la variable de entorno `CI` está configurada como `true`. Tanto [GitHub Actions][github actions] como [GitLab CI/CD][gitlab ci] configuran esta variable de manera predeterminada.

Conflictos con [la opción `--project`][project option].
