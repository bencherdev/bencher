### `--token <TOKEN>`

> 🐰 OBSOLÈTE : Les jetons API sont obsolètes au profit des clés API.
> De nouveaux jetons API ne peuvent plus être créés ; les jetons API existants continuent de fonctionner.
> Utilisez plutôt l'option `--key`.

Soit l'option `--token`, soit la variable d'environnement `BENCHER_API_TOKEN` peut être définie avec un jeton API valide. Si les deux sont spécifiés, l'option `--token` prévaut sur la variable d'environnement `BENCHER_API_TOKEN`.
