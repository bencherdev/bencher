## GitHub Actions

Installieren Sie die Bencher CLI mit der [GitHub Action](https://github.com/marketplace/actions/bencher-cli).
Weitere Details finden Sie unter [Github Actions verwenden](/docs/de/how-to/github-actions/).

<CliGitHubActions />

Es gibt ein optionales `version` Argument, um eine genaue Version der Bencher CLI zu verwenden.
Andernfalls wird standardmäßig die neueste CLI-Version verwendet.
Geben Sie eine genaue Version an, wenn und nur wenn Bencher *Self-Hosted* verwendet wird.
Geben Sie __nicht__ eine genaue Version an, wenn Sie Bencher *Cloud* verwenden, da es noch gelegentlich zu Kompatibilitätsproblemen kommt.
