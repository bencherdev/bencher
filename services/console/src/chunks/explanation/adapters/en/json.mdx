## \{...\} JSON

The JSON Adapter (`json`) expects [Bencher Metric Format (BMF) JSON][bmf].
It is perfect for integrating custom benchmark harnesses with Bencher.
For more details see [how to track custom benchmarks][custom benchmarks]
and the [BMF JSON reference][bmf].
