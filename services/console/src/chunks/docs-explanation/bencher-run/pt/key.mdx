### `--key <KEY>`

<br />

Opcional: A opção `--key` ou a variável de ambiente `BENCHER_API_KEY` pode ser definida com uma chave de API de projeto válida. Se ambas forem especificadas, a opção `--key` tem precedência sobre a variável de ambiente `BENCHER_API_KEY`. As chaves de API de projeto são limitadas a um único projeto e só podem ser usadas para `bencher run` e outros comandos não destrutivos. Conflita com [a opção `--token`](#--token-token).
