Bare Metal Runners execute benchmarks packaged in [OCI Image Format][oci].
When using the <code><a href="/docs/explanation/bencher-run/">bencher run</a></code> CLI subcommand,
specify an Image with the `--image` option to enable remote, bare metal execution.

When [Continuous Benchmarking][continuous benchmarking], that is benchmarking in CI,
using bare metal runners can drastically reduce the amount of noise in your benchmark results.
Further, Bencher Bare Metal can be run outside of CI using the _exact same_ bare metal runners.
This allows both developers and agents to check the performance of their code changes
without having to go through an entire CI workflow.
See the [Bare Metal Workflow][workflow] for a full overview.
