## Criar um Runner

Um Runner é o recurso que autentica o binário `runner` no seu servidor de API Bencher.
Crie um com a [CLI `bencher`][bencher runner cli],
a [Runners REST API][runners api],
ou o Bencher Console.

```sh
bencher runner create --name "My Runner"
```

A Bencher retorna o novo Runner junto com sua chave:

```json
{
  "uuid": "...",
  "key": "bencher_runner_..."
}
```

A chave é exibida apenas uma vez, então guarde-a com segurança.
Você a passará ao binário `runner` quando iniciar o Runner.
Se uma chave for perdida ou vazada, rotacione-a com `bencher runner key <RUNNER>`
(o endpoint [`POST /v0/runners/{runner}/key`][runners api]),
que invalida imediatamente a chave anterior.
