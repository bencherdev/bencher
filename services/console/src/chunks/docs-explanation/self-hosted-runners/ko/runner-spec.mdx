## Runner에 Spec 할당

Runner Spec은 [Spec][spec]을 Runner에 연결합니다.
하나의 Runner는 여러 Spec을 지원할 수 있고, 하나의 Spec은 여러 Runner가 지원할 수 있습니다.
Runner는 자신에게 할당된 Spec의 Job만 청구(claim)합니다.

[`bencher` CLI][runner specs api]
또는 [Runner Specs REST API][runner specs api]로 Runner에 Spec을 추가하세요:

```sh
bencher runner spec add my-runner --spec intel-v1
```

나중에 `bencher runner spec remove <RUNNER> <SPEC>`
([`DELETE /v0/runners/{runner}/specs/{spec}`][runner specs api] 엔드포인트)로 연결을 제거할 수 있습니다.
