## 🪄 Magic <small>(par défaut)</small>

L'adaptateur Magic (`magic`) est un sur-ensemble de tous les autres adaptateurs.
Pour cette raison, c'est l'adaptateur par défaut pour `bencher run`.
Cependant, l'adaptateur `magic` devrait être utilisé uniquement pour l'exploration.
