## Job

Job은 원격 벤치마크 실행 요청의 생명 주기를 추적합니다.
`--job-timeout` 옵션을 사용하여 Job별로 사용자 정의 타임아웃을 요청할 수 있습니다.
단, 다음 Job 타임아웃은 항상 적용됩니다:

| [티어][pricing] | Job 타임아웃                    |
| --------------- | ------------------------------- |
| Unclaimed       | 1분                             |
| Free            | 5분                             |
| Team            | 무제한 (기본값: 60분)           |
| Enterprise      | 무제한 (기본값: 60분)           |

전체 개요는 [Job 문서][job]를 참조하세요.
