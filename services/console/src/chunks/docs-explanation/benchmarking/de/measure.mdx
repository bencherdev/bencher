### Measure
Eine Measure ist die Maßeinheit für eine Metric.
Standardmäßig beginnen alle Projects mit einer `Latency`- und einer `Throughput`-Measure
mit den Einheiten `nanoseconds (ns)` bzw. `operations / second (ops/s)`.
Die Measure wird durch den [Benchmark-Harness-Adapter][adapters] bestimmt.

Wenn Sie das [Bencher Metric Format (BMF)][bmf] verwenden, können Name, Slug oder UUID verwendet werden, um eine Measure zu identifizieren.
