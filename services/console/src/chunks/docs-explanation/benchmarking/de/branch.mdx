### Branch
Ein Branch ist die `git`-Referenz, die beim Ausführen eines Berichts verwendet wird (d. h. Branch-Name oder Tag). Bei der Verwendung des <code><a href="/de/docs/explanation/bencher-run/">bencher run</a></code> CLI-Unterbefehls wird, wenn kein Branch angegeben ist, der aktuelle `git`-Branch-Name verwendet, falls verfügbar. Andernfalls wird `main` als Standard-Branch verwendet. Siehe [Branches](/de/docs/explanation/branches/) für eine vollständige Übersicht.
