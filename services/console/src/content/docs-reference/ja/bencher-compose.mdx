[Bencher Self-Hosted][bencher self hosted]をデプロイする最も簡単な方法は、`bencher` CLIのサブコマンド3つを使用することです：

- [`bencher up`][bencher up]: Bencher Self-Hostedのコンテナを起動します。
  これは[`docker compose up`][docker compose up]を実行するのと似ています。
- [`bencher logs`][bencher logs]: [ `--detach`フラグ][detach flag]を付けた`bencher up`が実行された場合、Bencher Self-Hostedコンテナの出力を表示します。
  これは[`docker compose logs`][docker compose logs]を実行するのと似ています。
- [`bencher down`][bencher down]: [ `--detach`フラグ][detach flag]を付けた`bencher up`が実行された場合、Bencher Self-Hostedのコンテナを停止して削除します。
  これは[`docker compose down`][docker compose down]を実行するのと似ています。
