### `--key <KEY>`

<br />

Необязательно: Опция `--key` или переменная окружения `BENCHER_API_KEY` может быть установлена в качестве действительного ключа API проекта. Если указаны оба, опция `--key` имеет приоритет над переменной окружения `BENCHER_API_KEY`. Ключи API проекта ограничены одним проектом и могут использоваться только для `bencher run` и других неразрушающих команд. Ключ API проекта нельзя использовать для создания нового Проекта или присвоения существующего Проекта. Конфликтует с [опцией `--token`](#--token-token).
