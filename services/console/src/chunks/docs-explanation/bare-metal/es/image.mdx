## Image

Una Image es una [imagen de contenedor OCI][oci image] que contiene tu código de benchmark y sus dependencias.
Cuando un Runner ejecuta un Job, descarga la Image, desempaqueta la Image y ejecuta el comando de benchmark especificado en el [Job][job].
Consulta [la documentación de Image][image] para saber cómo crear una Image para tu propio proyecto.
