## Spec

A Spec is a hardware specification that describes the resources available to a [Runner][runner].
A [Job][job] must specify which Spec it wants to use.
The Job is then assigned to a Runner that supports the specified Spec.
See [Testbeds and Specs][testbeds] for a full overview.
