[`runner` 二进制文件][runner cli] 和 API 服务器通过单个 WebSocket 连接通信。
本参考描述该协议：交换的消息、它们驱动的 Job 生命周期，
以及超时和重连如何防止 Job 卡住。它是 [Self-Hosted Runner][self hosted runners]
指南的深入补充。

使用 [`runner up`][runner up] 操作 Runner 并不需要了解这些内容；
提供它们是为了透明，也是为了任何围绕 API 构建工具的人。
