## 🎯 Dart benchmark_harness

Dart benchmark_harness 어댑터(`dart_benchmark_harness`)는 Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) 패키지의 출력(기본 `PrintEmitter` 라인: `Name(RunTime): <마이크로초> us.`)을 기대합니다.
`지연시간(latency)` 측정값(예: `나노초(ns)`)이 수집됩니다.
`보고된 값(reported value)`(예: `value`)만 사용 가능합니다. `하한값(lower_value)`과 `상한값(upper_value)`은 없습니다.
