## GitHub Actions

[GitHub Action](https://github.com/marketplace/actions/bencher-cli)を使用してBencher CLIをインストールします。
詳細は[GitHub Actionsの使用方法](/docs/ja/how-to/github-actions/)を参照してください。

<CliGitHubActions />

任意の`version`引数を使用してBencher CLIの具体的なバージョンを指定することができます。
それ以外の場合、最新のCLIバージョンがデフォルトとして使用されます。
特定のバージョンを指定するのは、Bencher *Self-Hosted* を使用している時だけです。
まだたまに互換性がない変更があるため、Bencher *Cloud* を使用している場合は具体的なバージョンを指定しないでください。
