### `--spec-reset`

<br />

Reset the [Testbed][testbed] [Spec][spec], removing its hardware specification.
This clears the `spec` associated with the Testbed when the run is submitted.
Requires [the `--testbed` option][testbed option].
Conflicts with [the `--image` option][image option].
