### `--key <KEY>`

<br />

선택사항: `--key` 옵션 또는 `BENCHER_API_KEY` 환경 변수를 유효한 프로젝트 API 키로 설정할 수 있습니다.
둘 다 지정된 경우 `--key` 옵션이 `BENCHER_API_KEY` 환경 변수보다 우선합니다.
프로젝트 API 키는 단일 프로젝트로 제한되며 `bencher run` 및 기타 비파괴적 명령에만 사용할 수 있습니다.
프로젝트 API 키는 새 프로젝트를 생성하거나 기존 프로젝트를 청구하는 데 사용할 수 없습니다.
[`--token` 옵션](#--token-token)과 충돌합니다.
