### `--ci-only-on-alert`

<br />

Opcional: Comece a enviar resultados para CI apenas se [um Alerta for gerado](/pt/docs/explanation/thresholds/#alerts). Se um Alerta for gerado, todos os resultados subsequentes também serão enviados, mesmo que não contenham nenhum Alerta. Isso se aplica apenas aos comentários no pull request. A Verificação do GitHub é sempre criada. Requer: `--github-actions`
