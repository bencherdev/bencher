### `plus.runners`

Этот раздел настраивает систему раннеров для выполнения бенчмарков на bare metal.
Раннеры — это серверные агенты, которые запрашивают и выполняют задачи бенчмарков.

Если этот раздел не настроен, Bencher будет использовать значения по умолчанию.

|           Имя             | Пример | По умолчанию | Обязательно |                                          Описание                                           |
| :-----------------------: | :----: | :----------: | :---------: | :-----------------------------------------------------------------------------------------: |
|    heartbeat_timeout      |   90   |      90      |     Нет     | Время в секундах без хартбита до пометки задачи как неудавшейся.                            |
| job_timeout_grace_period  |   60   |      60      |     Нет     | Дополнительное время в секундах сверх настроенного тайм-аута задачи до её отмены сервером.  |
