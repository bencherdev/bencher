## Image

Une Image est une [image de conteneur OCI][oci image] qui contient votre code de benchmark et ses dépendances.
Lorsqu'un Runner exécute un Job, il récupère l'Image, décompresse l'Image et exécute la commande de benchmark spécifiée dans le [Job][job].
Consultez [la documentation de l'Image][image] pour savoir comment créer une Image pour votre propre projet.
