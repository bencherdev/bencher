### `--image <IMAGE>`

<br />

Set the OCI image reference for remote [Bare Metal Runner][runner] execution
(e.g. `my-image:tag`, `registry.bencher.dev/my-image:tag`,
or for Bencher Self-Hosted instances `localhost:61016/my-image:tag`).
Only the OCI registry for the Bencher API server is supported.
When `--image` is set, the benchmark command runs on a remote Bare Metal Runner
instead of locally.

> 🐰 [Shared Bare Metal Runners][runner] do **not** have network access.
> That is, you cannot pull down any dependencies while executing your benchmarks.
> Your Image must be entirely self-contained.

[runner]: /docs/explanation/bare-metal/#runner
