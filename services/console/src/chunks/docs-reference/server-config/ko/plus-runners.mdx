### `plus.runners`

이 섹션은 베어메탈 벤치마크 실행을 위한 러너 시스템을 구성합니다.
러너는 서버 범위의 에이전트로 벤치마크 작업을 요청하고 실행합니다.

이 섹션이 구성되지 않은 경우, Bencher는 기본값을 사용합니다.

|           이름            | 예시 | 기본값 |  필수  |                                         설명                                          |
| :-----------------------: | :--: | :----: | :----: | :-----------------------------------------------------------------------------------: |
|    heartbeat_timeout      |  90  |   90   | 아니오 | 하트비트 없이 작업을 실패로 표시하기까지의 시간(초).                                  |
| job_timeout_grace_period  |  60  |   60   | 아니오 | 작업의 구성된 타임아웃을 초과하여 서버가 취소하기까지의 추가 시간(초).                |
