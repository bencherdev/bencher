### `--ci-only-thresholds`

<br />

Опционально: отправлять результаты в CI только если [существует пороговое значение](/ru/docs/explanation/thresholds/) для ветки, тестовой среды и меры.
Если пороговые значения отсутствуют, комментарий к запросу на вытягивание отправляться не будет.
Это относится только к комментариям к запросам на вытягивание. Проверка GitHub создается всегда.
Требуется: `--github-actions`
