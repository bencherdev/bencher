## Image

Uma Image é uma [imagem de contêiner OCI][oci image] que contém o seu código de benchmark e suas dependências.
Quando um Runner executa um Job, ele baixa a Image, descompacta a Image e executa o comando de benchmark especificado no [Job][job].
Consulte [a documentação da Image][image] para saber como criar uma Image para o seu próprio projeto.
