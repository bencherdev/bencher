### `plus.google`

This section specifies the configuration for [a Google client used for OAuth2 authentication](/docs/how-to/google-oauth/).
You must have a valid Bencher Plus _Enterprise_ license for at least one organization on the server.
The entire section is optional.
If not specified, then authentication with Google will not be enabled.

|     Name      |                         Example                          | Default | Required |                      Description                       |
| :-----------: | :------------------------------------------------------: | :-----: | :------: | :----------------------------------------------------: |
|   client_id   | 0123456789-abcdefg0112358envs.apps.googleusercontent.com |   ---   |   Yes    | Specifies the client ID for your Google OAuth2 Client. |
| client_secret |                   GOCSPX-xyz987654321                    |   ---   |   Yes    |    The client secret for your Google OAuth2 Client.    |
