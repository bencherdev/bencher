## `bencher mock`

`bencher mock` CLIサブコマンドは、モックの Bencher Metric Format (BMF) JSON データを生成するために使用します。

### `--seed <SEED>`

疑似乱数生成器のシードです。
指定しない場合はランダムなシードが使用されます。

### `--count <COUNT>`

生成するモックの[Benchmarks][benchmark]の数。

[benchmark]: /ja/docs/explanation/benchmarking/#benchmark

### `--measure <MEASURE>`

各[Benchmark][benchmark]に使用する[Measure][measure]。
`--measure`オプションを複数回使用することで、複数の Measure を指定できます。
デフォルトは組み込みの `latency` Measure です。

[measure]: /ja/docs/explanation/benchmarking/#measure
[benchmark]: /ja/docs/explanation/benchmarking/#benchmark

### `--pow <POW>`

モックの[Metrics][metric]に使用する 10 の冪指数。
デフォルトは `1` です。

[metric]: /ja/docs/explanation/benchmarking/#metric

### `--fail`

実行中に失敗します。

### `--flaky`

実行中に断続的に失敗します。

### `--help`

ヘルプメッセージを表示します。
