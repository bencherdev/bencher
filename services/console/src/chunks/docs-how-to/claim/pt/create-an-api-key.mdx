## Criar uma Chave de API

Para usar o
<code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code> subcomando CLI
com a [opção `--key`][key option],
você precisará criar uma chave de API de usuário.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9wdC9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">Clique para adicionar uma Chave de API</a></h3>

### Etapas:

1. Navegue até o Console Bencher.
2. Passe o mouse sobre seu nome no canto superior direito.
3. Um menu suspenso deve aparecer. Selecione `Keys`.
4. Na página de Chaves de API, clique no botão `➕ Add`.

[key option]: /pt/docs/explanation/bencher-run/#--key-key
