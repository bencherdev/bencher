## Einen Runner erstellen

Ein Runner ist die Ressource, die das `runner`-Binary gegenüber Ihrem Bencher API-Server authentifiziert.
Erstellen Sie einen mit der [`bencher` CLI][bencher runner cli],
der [Runners REST-API][runners api]
oder der Bencher Console.

```sh
bencher runner create --name "My Runner"
```

Bencher gibt den neuen Runner zusammen mit seinem Schlüssel zurück:

```json
{
  "uuid": "...",
  "key": "bencher_runner_..."
}
```

Der Schlüssel wird nur einmal angezeigt, bewahren Sie ihn also sicher auf.
Sie übergeben ihn dem `runner`-Binary, wenn Sie den Runner starten.
Falls ein Schlüssel jemals verloren geht oder durchsickert, rotieren Sie ihn mit `bencher runner key <RUNNER>`
(dem Endpunkt [`POST /v0/runners/{runner}/key`][runners api]),
was den vorherigen Schlüssel sofort ungültig macht.
