Интерфейсу Bencher Console не требуется никакой настройки.
Однако некоторые параметры по умолчанию можно изменить под ваши нужды
с помощью следующих переменных окружения:

- `BENCHER_API_URL`: URL для [сервера Bencher API][server config].
  По умолчанию установлено значение `http://localhost:61016`.
- `INTERNAL_API_URL`: При использовании сетевой конфигурации Docker, например с `docker compose`,
  это необходимо для связи контейнеров Docker при использовании локального `BENCHER_API_URL`.
  Не устанавливайте эту переменную при использовании удалённого `BENCHER_API_URL`, например `https://bencher.example.com`.
  По умолчанию не задана.
- `OAUTH_GITHUB`: Установите в `true`, чтобы включить `Login with GitHub` с использованием [локально размещённого приложения GitHub][github app].
  Это требует [плана Bencher Plus Enterprise][bencher plus enterprise]. По умолчанию не задана.
- `OAUTH_GOOGLE`: Установите в `true`, чтобы включить `Login with Google` с использованием [локально размещённого Google OAuth2][google oauth2].
  Это требует [плана Bencher Plus Enterprise][bencher plus enterprise]. По умолчанию не задана.
