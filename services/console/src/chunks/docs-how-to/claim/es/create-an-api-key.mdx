## Crear una Clave de API

Para usar el
<code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code> subcomando de CLI
con la [opción `--key`][key option],
necesitarás crear una clave de API de usuario.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9lcy9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">Haz clic para añadir una Clave de API</a></h3>

### Pasos:

1. Navega a la Consola de Bencher.
2. Pasa el cursor sobre tu nombre en la esquina superior derecha.
3. Debería aparecer un menú desplegable. Selecciona `Keys`.
4. Una vez en la página de Claves de API, haz clic en el botón `➕ Add`.

[key option]: /es/docs/explanation/bencher-run/#--key-key
