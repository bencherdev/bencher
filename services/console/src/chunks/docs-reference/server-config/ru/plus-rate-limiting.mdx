### `plus.rate_limiting`

Этот раздел определяет ограничения скорости для сервера API.
Весь раздел является необязательным.
Если не указано, будут использоваться перечисленные значения по умолчанию.

Чтобы предотвратить создание [`unclaimed` проектов][unclaimed projects],
установите `unclaimed_limit` на `0`.

|          Имя           |    Пример    | По умолчанию | Обязательно |                                                                               Описание                                                                               |
| :--------------------: | :----------: | :----------: | :---------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|         window         |    86400     |    86400     |     Нет     |                                       Определяет окно ограничения скорости в секундах. По умолчанию установлено на один день.                                        |
|    unclaimed_limit     |     255      |     255      |     Нет     | Контролирует ограничение скорости для [`unclaimed` проектов][unclaimed projects]. Чтобы предотвратить создание `unclaimed` проектов, установите это значение на `0`. |
|     claimed_limit      |    65536     |    65536     |     Нет     |                                                      Контролирует ограничение скорости для `claimed` проектов.                                                       |
|     oci_bandwidth      |  \{ ... \}   |     ---      |     Нет     |                            Определяет дневные лимиты пропускной способности OCI-реестра для каждого уровня организации.                                               |
| oci_bandwidth.unclaimed | 1073741824  | 1073741824   |     Нет     |                          Максимум байт в день для невостребованных организаций (0 участников). По умолчанию 1 ГиБ.                                                    |
|   oci_bandwidth.free   | 10737418240  | 10737418240  |     Нет     |                          Максимум байт в день для бесплатных (востребованных, без платного плана) организаций. По умолчанию 10 ГиБ.                                   |
|   oci_bandwidth.plus   | 107374182400 | 107374182400 |     Нет     |                          Максимум байт в день для организаций Plus (Team/Enterprise). По умолчанию 100 ГиБ.                                                          |

[unclaimed projects]: /ru/docs/how-to/claim/#unclaimed-projects
