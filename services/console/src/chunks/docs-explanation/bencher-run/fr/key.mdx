### `--key <KEY>`

<br />

Optionnel : L'option `--key` ou la variable d'environnement `BENCHER_API_KEY` peut etre definie avec une cle API de projet valide. Si les deux sont specifiees, l'option `--key` a la priorite sur la variable d'environnement `BENCHER_API_KEY`. Les cles API de projet sont limitees a un seul projet et ne peuvent etre utilisees que pour `bencher run` et d'autres commandes non destructives. En conflit avec [l'option `--token`](#--token-token).
