#### Начальная точка
Ветка может иметь [начальную точку][start point].
Начальная точка — это другая ветка на определенной версии (и `git` хэш, если доступно).
Исторические метрики и, по желанию, пороговые значения копируются из начальной точки.
См. [ветки](/ru/docs/explanation/branches/) для полного обзора.

[start point]: https://git-scm.com/docs/git-branch#Documentation/git-branch.txt-ltstart-pointgt
