### Mesure
Une Mesure est l'unité de mesure d'une Métrique.
Par défaut, tous les projets démarrent avec des Mesures `Latency` et `Throughput`
avec des unités de `nanoseconds (ns)` et `operations / second (ops/s)` respectivement.
La Mesure est déterminée par l'[adaptateur du moteur de benchmark][adapters].

Si vous utilisez le [Bencher Metric Format (BMF)][bmf], le nom, le slug ou l'UUID peuvent être utilisés pour identifier une Mesure.
