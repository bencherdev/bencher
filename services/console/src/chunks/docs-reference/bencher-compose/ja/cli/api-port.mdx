### `--api-port <PORT>`

API サーバーコンテナのポート番号を指定します。
デフォルトではポート `61016` が使用されます。
これは `docker run` における [`--expose` オプション][docker run expose] のポートのみの形式に類似しています。

`--api-port` オプションを指定した場合は、[Console Server の `BENCHER_API_URL` 環境変数][console server] を [`--console-env` オプション][console env] を使って新しいポートに合わせて更新する必要があります。
