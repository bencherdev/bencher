#### Spec
Spec — это спецификация оборудования, описывающая ресурсы, доступные для испытательной платформы.
Она включает архитектуру CPU, количество CPU, объём памяти, объём диска и наличие сетевого доступа в среде.
Spec имеют серверную область видимости, поэтому одна и та же Spec может использоваться несколькими испытательными платформами.
