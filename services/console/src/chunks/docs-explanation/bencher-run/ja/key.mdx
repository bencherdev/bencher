### `--key <KEY>`

<br />

オプション: `--key` オプションまたは `BENCHER_API_KEY` 環境変数のいずれかを設定して、有効なプロジェクトAPIキーを指定できます。
両方が指定されている場合、`--key` オプションが `BENCHER_API_KEY` 環境変数よりも優先されます。
プロジェクトAPIキーは単一のプロジェクトに限定され、`bencher run` およびその他の非破壊的なコマンドにのみ使用できます。
プロジェクトAPIキーは、新しい Project の作成や既存の Project の取得には使用できません。
[`--token` オプション](#--token-token)と競合します。
