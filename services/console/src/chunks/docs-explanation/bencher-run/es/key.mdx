### `--key <KEY>`

<br />

Opcional: La opcion `--key` o la variable de entorno `BENCHER_API_KEY` puede establecerse con una clave de API de proyecto valida. Si ambas se especifican, la opcion `--key` tiene precedencia sobre la variable de entorno `BENCHER_API_KEY`. Las claves de API de proyecto estan limitadas a un solo proyecto y solo pueden usarse para `bencher run` y otros comandos no destructivos. Entra en conflicto con [la opcion `--token`](#--token-token).
