### `--spec-reset`

<br />

➕ Bencher Plus: Setzt die [Testbed][testbed]-[Spec][spec] zurück und entfernt die Hardwarespezifikation.
Dies löscht die mit dem Testbed verknüpfte `spec`, wenn der Lauf übermittelt wird.
Erforderlich: `--testbed`
Konflikte mit: `--image`
