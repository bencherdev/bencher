## API 키 생성하기

<code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code> CLI 하위 명령어를 [the `--key` 옵션][key option]과 함께 사용하려면, 사용자 API 키를 생성해야 합니다.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9rby9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">API 키 추가를 위해 클릭하세요</a></h3>

### 단계:

1. Bencher 콘솔로 이동하세요.
2. 오른쪽 상단에 있는 자신의 이름을 마우스로 가리키세요.
3. 드롭다운 메뉴가 나타납니다. `Keys`를 선택하세요.
4. API 키 페이지에 도착하면, `➕ Add` 버튼을 클릭하세요.

[key option]: /ko/docs/explanation/bencher-run/#--key-key
