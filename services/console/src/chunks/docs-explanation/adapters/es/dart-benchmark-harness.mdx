## 🎯 Dart benchmark_harness

El adaptador Dart benchmark_harness (`dart_benchmark_harness`) espera la salida del paquete Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) (líneas predeterminadas de `PrintEmitter`: `Name(RunTime): <microsegundos> us.`).
Se recopila la medida de `latencia` (es decir, `nanosegundos (ns)`).
Solo está disponible el valor reportado (es decir, `value`). No hay `lower_value` y `upper_value`.
