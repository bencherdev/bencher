## Создание API ключа

Для использования
<code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code> подкоманды CLI
с [опцией `--key`][key option],
вам необходимо создать ключ API пользователя.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9ydS9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">Кликните, чтобы добавить API ключ</a></h3>

### Шаги:

1. Перейдите в Bencher Console.
2. Наведите курсор на ваше имя в правом верхнем углу.
3. Должно появиться выпадающее меню. Выберите `Keys`.
4. На странице API Keys нажмите кнопку `➕ Add`.

[key option]: /ru/docs/explanation/bencher-run/#--key-key
