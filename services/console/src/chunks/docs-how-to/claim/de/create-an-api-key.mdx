## API-Schlüssel erstellen

Um den
<code><a href="/de/docs/explanation/bencher-run/">bencher run</a></code>-CLI-Unterbefehl
mit [der `--key` Option][key option]
zu verwenden, müssen Sie einen Benutzer-API-Schlüssel erstellen.

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kZS9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">Klicken Sie hier, um einen API-Schlüssel hinzuzufügen</a></h3>

### Schritte:

1. Navigieren Sie zur Bencher-Konsole.
2. Bewegen Sie den Mauszeiger über Ihren Namen in der oberen rechten Ecke.
3. Ein Dropdown-Menü sollte erscheinen. Wählen Sie `Keys`.
4. Sobald Sie auf der Seite API-Schlüssel sind, klicken Sie auf die Schaltfläche `➕ Add`.

[key option]: /de/docs/explanation/bencher-run/#--key-key
