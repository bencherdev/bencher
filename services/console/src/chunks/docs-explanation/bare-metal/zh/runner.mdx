## Runner

Runner 是一个在专用裸金属硬件上运行的远程基准测试执行器。
Runner 根据其关联的 [Specs][spec] 轮询 Bencher API 服务器以获取待处理的 [Jobs][job]。
一旦认领了一个 Job，Runner 会拉取该 Job 指定的 [Image][image]，
解包 Image，执行基准测试命令，并将结果发送回 Bencher API 服务器。
每次只执行一个 Job，以确保基准测试不受其他工作负载的影响。
