## APIキーを作成する

<code><a href="/ja/docs/explanation/bencher-run/">bencher run</a></code> CLIサブコマンドを
[`--key`オプション][key option]で使用するには、
ユーザーAPIキーを作成する必要があります。

<h3><a id="add-api-key" href="/console/users/keys/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9qYS9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS1rZXk=">APIキーを追加するにはここをクリック</a></h3>

### 手順:

1. Bencherコンソールに移動します。
2. 右上の自分の名前にマウスを合わせます。
3. ドロップダウンメニューが表示されます。`Keys`を選択します。
4. APIキーのページで、`➕ Add`ボタンをクリックします。

[key option]: /ja/docs/explanation/bencher-run/#--key-key
