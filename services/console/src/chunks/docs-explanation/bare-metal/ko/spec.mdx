## Spec

Spec은 [Runner][runner]에서 사용 가능한 리소스를 설명하는 하드웨어 사양입니다:

1. 운영 체제 (OS)
   1. Linux
   2. macOS
   3. Windows
2. CPU 아키텍처
   1. x86_64
   2. ARM64
3. Sandbox
   1. Firecracker microVM
   2. Sandbox 없음
4. CPU 수
5. 메모리 크기 (RAM)
6. 디스크 크기
7. 네트워크 접근

Spec은 서버 범위 또는 조직 범위로 설정되며,
동일한 Spec을 같은 서버의 여러 [Testbed][testbed]에서 공유하거나 같은 조직 내에서 공유할 수 있습니다.

<code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code> 사용 시 Spec 선택 우선순위는 다음과 같습니다:
1. `--spec` 옵션
2. Testbed에 할당된 Spec
3. 기본 조직 범위 Spec
4. 기본 서버 범위 Spec

선택된 Spec은 현재 [Testbed][testbed]에 할당됩니다.
