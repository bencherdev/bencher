#### Point de départ
Une branche peut avoir un [Point de départ][start point].
Un Point de départ est une autre branche à une version spécifique (et un `git` hash, si disponible).
Les métriques historiques et éventuellement les seuils sont copiés depuis le Point de départ.
Voir [les branches](/fr/docs/explanation/branches/) pour une vue d'ensemble complète.

[start point]: https://git-scm.com/docs/git-branch#Documentation/git-branch.txt-ltstart-pointgt
