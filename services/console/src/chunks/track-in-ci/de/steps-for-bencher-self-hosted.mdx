## Schritte für die Bencher-Self-Hosted

1. Erstellen Sie eine [Bencher Self-Hosted Instanz][bencher self hosted].
2. Erstellen Sie ein Konto auf Ihrer Bencher Self-Hosted Instanz.
3. Erstellen Sie einen Benutzer-API-Schlüssel und fügen Sie ihn als Geheimnis zu Ihrem CI hinzu.
4. Erstellen Sie einen Workflow für Ihr CI, wie [GitHub Actions][github actions] oder [GitLab CI/CD][gitlab ci cd].
5. [Installieren Sie die Bencher-CLI][install bencher cli] in Ihrem CI-Workflow.
   Stellen Sie sicher, dass die CLI-Version mit der Version Ihrer Bencher Self-Hosted Instanz übereinstimmt.
6. {props.step}
