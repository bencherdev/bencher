### `--token <TOKEN>`

<br />

> 🐰 OBSOLÈTE : Les jetons API sont obsolètes au profit des clés API.
> De nouveaux jetons API ne peuvent plus être créés ; les jetons API existants continuent de fonctionner.
> Utilisez plutôt [l'option `--key`](#--key-key).

Optionnel : Soit l'option `--token`, soit la variable d'environnement `BENCHER_API_TOKEN`
peut être configurée avec un jeton API valide.
Si les deux sont spécifiés, l'option `--token` prévaut sur la variable d'environnement `BENCHER_API_TOKEN`.
Si ni l'option `--token` ni [l'option `--key`](#--key-key) ne sont spécifiées, le Projet doit être `non réclamé`.
C'est-à-dire que si vous avez déjà `réclamé` un Projet, vous devez fournir une clé API ou un jeton API valide.
En conflit avec [l'option `--key`](#--key-key).
