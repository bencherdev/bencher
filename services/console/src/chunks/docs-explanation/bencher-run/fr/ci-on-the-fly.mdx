### `--ci-on-the-fly`

<br />

Optionnel : Autoriser la création de [projets à la volée][on the fly projects] dans les environnements CI.
Requis si la variable d'environnement `CI` est définie sur `true`.
Tant [GitHub Actions][github actions] que [GitLab CI/CD][gitlab ci] définissent cette variable par défaut.

En conflit avec [l'option `--project`][project option].
