### `plus.google`

Cette section spécifie la configuration pour [un client Google utilisé pour l'authentification OAuth2](/fr/docs/how-to/google-oauth/).
Vous devez disposer d'une licence Bencher Plus _Entreprise_ valide pour au moins une organisation sur le serveur.
Toute la section est facultative.
Si elle n'est pas spécifiée, l'authentification via Google ne sera pas activée.

|      Nom      |                         Exemple                          | Par défaut | Obligatoire |                   Description                   |
| :-----------: | :------------------------------------------------------: | :--------: | :---------: | :---------------------------------------------: |
|   client_id   | 0123456789-abcdefg0112358envs.apps.googleusercontent.com |    ---     |     Oui     | Spécifie l'identifiant du client OAuth2 Google. |
| client_secret |                   GOCSPX-xyz987654321                    |    ---     |     Oui     |       Le secret du client OAuth2 Google.        |
