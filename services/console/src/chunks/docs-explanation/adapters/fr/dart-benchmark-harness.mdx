## 🎯 Dart benchmark_harness

L'adaptateur Dart benchmark_harness (`dart_benchmark_harness`) attend la sortie du package Dart [`benchmark_harness`](https://pub.dev/packages/benchmark_harness) (lignes `PrintEmitter` par défaut : `Name(RunTime): <microsecondes> us.`).
La mesure de `latence` (c'est-à-dire `nanosecondes (ns)`) est recueillie.
Seule la valeur rapportée (c'est-à-dire `valeur`) est disponible. Il n'y a pas de `lower_value` et de `upper_value`.
